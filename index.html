Welcome

Esta página tiene por objetivo mostrar los proyectos realizados en Imagen Escrita 2020.

Pizarra

Esta fue la primera idea de proyecto. Consiste en un programa de dibujo donde se pueden cambiar lo colores y grosores de las lineas. Se puede jugar con las teclas 0, 1, 2, 3, n, e, b, y, r, o, p, g y la barra espaceadora para hacer los dibujos.

Ir al proyecto »

Ciclo For

Este fue un primer acercamiento a lo que es el ciclo for. Al crear este proyecto el objetivo fue ver que ocurría al crear objetos y modificarlos, mientras que al mismo tiempor se iban cambiando los numeros del ciclo. Al final se llegó a una especie de trama de lineas las cuales van unidas entre sí.

Slider

Para este proyecto se quiso intentar algo que interactuara un poco mas con el usuario. Con formas básicas coloreadas y puestas en sus lugares, el primer acercamiento con los sliders crean un rostro el cual va abriendo y cerrando la boca.

Pizarra Morada

Este proyecto fue el primero de este tipo. Se me ocurrió hacer una herramienta que dibujara con colores aleatorios dentro de un rango (colpres morados). Al presionar la tecla "n" el color cambiará a uno aleatorio, y al preionar la barra espaceadora el dibujo se borrará.

Autorretrato

En este proyecto se utilizaron solo vectores, colores y transparencias para crear un autorretrato. Para esto se usó de referencia una fotografía que luego se calcó en físico. Luego se simplificaron las líneas, se la colocó en una grilla y se le asignaron valores a los nodos. A partir de estos valores se fue armando el autorretrato.
